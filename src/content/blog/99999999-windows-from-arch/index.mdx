ISOをダウンロード
マウント

```
pacman -S wimlib chntpw
```

```
wiminfo /path/to/install.wim
wimapply /path/to/install.wim 3 /dev/sdXY
```

Windowsが起動さえすればあとはそこから`bcdboot`がしばけるようになるので、最低限のBCDエントリを作成
